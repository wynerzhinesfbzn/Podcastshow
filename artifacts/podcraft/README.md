# PodCraft — AI Подкаст Генератор

Сервис автоматически превращает любой текст в полноценный подкаст с озвучкой, видео и презентацией.

## Что умеет PodCraft

- **Диалог**: GPT генерирует живой разговор двух ведущих из вашего текста
- **Озвучка**: OpenAI TTS озвучивает каждую реплику выбранными голосами
- **MP3**: все реплики склеиваются в один аудиофайл с фоновой музыкой и ducking-эффектом
- **MP4**: видео с AI-иллюстрациями (DALL-E 3) в выбранном стиле
- **PPTX**: готовая презентация с изображениями и тезисами

## Как запустить

### Требования

- Python 3.11+
- ffmpeg (уже установлен в Replit)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск

```bash
cd artifacts/podcraft
PORT=5000 python app.py
```

Или через Replit — воркфлоу **PodCraft Flask App** запускается автоматически.

## Использование

1. Откройте приложение в браузере
2. Введите текст (или загрузите .txt / .pdf / .docx, или вставьте ссылку на статью)
3. Выберите стиль изображений: аниме, кино, мультфильм, футуризм, инфографика
4. Выберите голоса ведущего и гостя из OpenAI TTS
5. Установите примерную длительность (1–15 минут)
6. Выберите фоновую музыку
7. Нажмите **«Создать подкаст»**
8. Дождитесь результата — прогресс отображается в реальном времени

### Вкладки результатов

- **🎧 Аудио** — MP3 плеер + скачать файл
- **🎬 Видео** — MP4 плеер + скачать + сменить музыку на лету
- **📊 Презентация** — скачать PPTX

## AI интеграции (через Replit)

- **GPT-5-mini** — генерация диалога
- **OpenAI TTS-1** — синтез речи (голоса: alloy, nova, echo, fable, onyx, shimmer)
- **DALL-E 3 / gpt-image-1** — генерация изображений для каждого сегмента

Ключи API предоставляются автоматически через Replit AI Integrations — вводить ничего не нужно.

## Технический стек

- **Flask 3** — веб-сервер
- **pydub** — склейка аудио + ducking
- **python-pptx** — создание PPTX-презентаций
- **ffmpeg** — сборка MP4 видео
- **trafilatura** — извлечение текста из веб-страниц
- **PyPDF2 / python-docx** — чтение PDF и DOCX файлов

## Переменные окружения

| Переменная | Описание |
|---|---|
| `OPENAI_API_KEY` | Автоматически от Replit AI Integrations |
| `OPENAI_BASE_URL` | Прокси Replit для OpenAI API |
| `SESSION_SECRET` | Секрет Flask сессий |
| `PORT` | Порт сервера (по умолчанию 5000) |
